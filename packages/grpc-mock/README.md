> 提供grpc mock数据自动生成和mock服务启动功能，减轻手动编写mock数据代码，mock数据填充默认值。支持mock数据更改时热更新获取新数据。

## 安装:

```bash
yarn add @liangskyli/grpc-mock --dev
```
